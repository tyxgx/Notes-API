    # 🗒️ Notes API

A RESTful API for managing user notes with:

- ✅ JWT-based authentication
- 👥 Role-Based Access Control (RBAC)
- 📄 Pagination support for listing notes
- 🔐 Secure password hashing with bcrypt

---

## 🚀 Tech Stack

- Node.js + Express
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- bcryptjs
